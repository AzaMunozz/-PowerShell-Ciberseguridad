# PowerShell - Ciberseguridad

## Desarrollador
Irving Azael Muñoz Ochoa

## Descripción del repositorio
Este repositorio contiene ejercicios desarrollados en PowerShell relacionados con tareas básicas de ciberseguridad.

## Scripts incluidos
- Scripts de PowerShell (.ps1)
- Módulos de PowerShell (.psm1)
- Manifiestos de módulos (.psd1)

## Tareas de ciberseguridad que resuelven
Los scripts permiten analizar conexiones de red, investigar direcciones IP externas y generar reportes de posibles riesgos de seguridad.

## ¿Qué aprendí?
Durante el desarrollo de estos ejercicios aprendí a utilizar PowerShell para automatizar tareas de seguridad, crear módulos y utilizar servicios externos para analizar información de red.
